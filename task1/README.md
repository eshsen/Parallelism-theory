# Вычисление суммы синусов  
## Описание задачи  
Заполнить масив float/double значениями синуса (один период на всю длину массива), количество элементов 10^7. Посчитать сумму и вывести в терминал.
## Сборка программы  
В папке <mark>task1</mark>:  

float:
```
cmake -S . -B build ИЛИ cmake -S . -B build -DUSE_DOUBLE=OFF
```
double:  
```
cmake -S . -B build -DUSE_DOUBLE=ON
```
## Результаты программы  
Ожидаемый вывод - 0.0
- Вывод с float - Sum: 0.094364
- Вывод с double - Sum: 4.89582e-11
